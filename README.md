# NYC-Collisions-Analysis

# Power BI Capstone Project

## Project Overview

This project analyzes **Motor Vehicle Collision** data reported by the **New York City**. The dashboard was developed using **Microsoft Power BI** to uncover accident trends, identify major contributing factors, and provide actionable insights into road safety in New York City.


## Project Objectives

The analysis answers the following business questions:

1. What percentage of total accidents occurred each month, and are there any seasonal patterns?
2. Which day of the week and hour of the day record the highest accident frequency?
3. Which street has the highest number of reported accidents, and what percentage of total accidents does it represent?
4. What are the most common contributing factors for all accidents and fatal accidents?

## Tools Used

* Microsoft Power BI
* Power Query
* DAX (Data Analysis Expressions)
* CSV Dataset


## Data Cleaning

The dataset was cleaned and transformed in Power Query by:

* Removing duplicates
* Handling missing values
* Correcting data types
* Creating date-related fields
* Standardizing text values
* Preparing the dataset for analysis


## Dashboard Features

The dashboard includes:

* KPI Cards

  * Total Accidents
  * Total Injured
  * Total Killed
  * Fatal Accidents

* Percentage of Total Accidents by Month

* Accidents by Day of the Week

* Accidents by Hour of the Day

* Top 10 Streets by Number of Accidents

* Top Contributing Factors (All Accidents)

* Top Contributing Factors (Fatal Accidents)

* Interactive Slicers (Borough, Month, Day of Week)


## Key Insights

* Approximately **238,000 accidents** were recorded.
* Around **116,000 people were injured**.
* **633 fatalities** were reported.
* **611 fatal accidents** occurred.
* **March** recorded the highest percentage of accidents.
* **Friday** experienced the highest accident frequency.
* Accidents peaked during the **late afternoon and evening hours**.
* **Belt Parkway** recorded the highest number of accidents.
* **Driver Inattention/Distraction**, **Unsafe Speed**, and **Unspecified** were among the leading contributing factors.


## Recommendations

* Increase traffic enforcement during peak commuting hours.
* Strengthen campaigns against distracted driving and speeding.
* Improve road safety measures on high-risk roads such as Belt Parkway.
* Enhance accident reporting to reduce the number of "Unspecified" contributing factors.
